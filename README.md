## 電影介紹網站
  使用 node.JS 平台搭配後端 express 框架建立電影介紹網站, 前端則是使用bootstrap框架, 內容涵蓋電影介紹, 熱門電影搜索...等
  
### 專案畫面
![image](https://github.com/LettuceLu751103/movie_list_remote/blob/master/movie_list_p1.png)

### 產品功能
1. 使用者可以瀏覽熱門電影
2. 使用者可以搜尋熱門電影
3. 使用者可以查看熱門電影詳情

### 建置環境
1. Node.JS : v10.15.0
2. Express : 4.17.1
3. express-handlebars : 5.3.0
4. Bootstrap : 4.2.1
